# Wat houdt het open-data project?
In dit project maken we statistieken en grafieken zichtbaar, gebaseerd op openbaar beschikbare datasets.
Deze inzichten zijn te doorzoeken via https://searchdata.com/ .

# Hoe werkt deze repository?
Deze repo gebruiken we voornamelijk om versiebeheer van scripts te realiseren. Voorheen stonden deze scripts alleen in de roosterstappen van de portalen.
Daarnaast is het mogelijk om kleine datasets toe te voegen in de folder data-bestanden.
